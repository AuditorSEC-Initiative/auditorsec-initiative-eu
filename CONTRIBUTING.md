# Contributing to AuditorSEC Initiative EU

Thank you for your interest in contributing to Europe’s Web3 AI Security Core!

## Ways to Contribute

- **Bug reports** — Open an issue using the Bug Report template.
- **Feature requests** — Open an issue using the Feature Request template.
- **Research contributions** — Academic papers, threat analyses, case studies.
- **Policy documents** — Regulatory alignment docs (MiCA, NIS2, DORA, Horizon).
- **Technical docs** — Architecture diagrams, integration guides.
- **Translations** — Ukrainian/English/EU language localization.
- **Partner nodes** — Become a pilot site or consortium partner.

## Repository Structure

```

│   └── grants/            # Grant proposals and one-pagers
├── programs/              # Specific program materials
│   ├── horizon-europe/    # Horizon Europe concept notes
│   ├── brave1/            # BRAVE1 materials
│   └── usf-edge/          # USF EDGE Diia.City materials
├── CONTRIBUTING.md
├── SECURITY.md
└── README.md
```

## How to Contribute Documentation

1. Fork the repository and create a branch: `git checkout -b docs/your-contribution`.
2. Add or update Markdown files in the appropriate `docs/` subdirectory.
3. Follow the document structure: Title, Overview, Sections, References.
4. Submit a Pull Request with a brief description of the contribution.
5. Await review within 5 business days.

## Research Contributions

- Peer-reviewed research relevant to Web3 security, PQC, or EU AI Act compliance is welcome.
- Include abstract, methodology, and findings summary.
- Link to full paper or preprint if available.

## Partner Program

Organizations interested in joining as consortium partners or pilot sites:

1. Open a [Partner Interest issue](https://github.com/AuditorSEC-Initiative/auditorsec-initiative-eu/issues/new).
2. Describe your organization, region, and area of interest.
3. Our partnerships team will respond within 5 business days.

## Code of Conduct

All contributors are expected to be professional, respectful, and act in the public interest. The AuditorSEC Initiative is committed to an inclusive, diverse contributor community.
